# 🏠 PG Manager Pro - Complete Tenant Management System

A professional, feature-rich web application for PG/Hostel owners to manage tenants, track payments, and share announcements through an integrated blog system.

## ✨ Features

### 🏢 Tenant Management
- ✅ Add, Edit, Delete tenant information
- ✅ Track tenant details (Name, Room, Mobile, Rent, Deposit)
- ✅ Search tenants by name, room, or mobile number
- ✅ Filter by active/inactive status
- ✅ View recent tenants on dashboard

### 💰 Payment Tracking
- ✅ Monthly fee tracking (Paid/Pending status)
- ✅ Filter payments by month and status
- ✅ Real-time payment status updates
- ✅ View pending payments dashboard
- ✅ Track monthly revenue

### 📝 Blog System
- ✅ Create, edit, delete blog posts
- ✅ Categorize posts (Announcements, Rules, Events, Maintenance, General)
- ✅ Search blog posts
- ✅ Rich text content support
- ✅ View count tracking
- ✅ Like system for posts
- ✅ Comment system with user names
- ✅ Tags support for better organization

### 📊 Reports & Analytics
- ✅ Export data to Excel (CSV format)
- ✅ Export reports to PDF
- ✅ Financial summary dashboard
- ✅ Collection rate calculation
- ✅ Total expected vs collected rent

### 💾 Data Persistence
- ✅ All data saved in browser's Local Storage
- ✅ No database required
- ✅ Works offline
- ✅ Data persists after browser refresh

### 📱 Responsive Design
- ✅ Mobile-friendly interface
- ✅ Tablet optimized
- ✅ Desktop ready
- ✅ Touch-friendly buttons
- ✅ Responsive tables with horizontal scroll

## 🔧 Installation

### Method 1: Direct Download
1. Download all three files:
   - `index.html`
   - `style.css`
   - `app.js`
2. Place them in the same folder
3. Open `index.html` in any modern web browser
